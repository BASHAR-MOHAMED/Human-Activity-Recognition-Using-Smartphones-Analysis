# Human Activity Recognition Using Smartphones Analysis

## Project Overview
This project processes the Human Activity Recognition Using Smartphones dataset to create a tidy dataset suitable for analysis.

## Files Description

### Script Files
- `run_analysis.R`: Main analysis script that processes the raw data

### Documentation Files
- `Codebook.md`: Describes variables, data, and transformations
- `README.md`: This file - explains how the scripts work

### Output Files
- `tidy_data.txt`: Final tidy dataset (created by the script)

## How to Run the Analysis

### Prerequisites
- R installed (version 4.0.0 or higher recommended)
- Internet connection (for downloading data)

### Required R Packages
- dplyr
- tidyr

These packages will be installed automatically if not present.

1. **Clone or download** this repository to your local machine
